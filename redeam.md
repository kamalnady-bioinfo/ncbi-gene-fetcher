# NCBI Gene Fetcher 🧬

A Python script to fetch real gene sequences from NCBI database using Biopython.

## Features
- Fetch gene sequences from NCBI
- Calculate GC content
- Display first 50 nucleotides

## Requirements
- Python 3
- Biopython

## Installation
pip install biopython

## Usage
python fetcher.py
